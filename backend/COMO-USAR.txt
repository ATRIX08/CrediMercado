Como iniciar o backend com Supabase/PostgreSQL:

1. Crie o arquivo backend/.env usando backend/.env.example.
2. Coloque a DATABASE_URL do Supabase no .env.
3. Instale as dependencias:

   cd backend
   npm install

4. Crie as tabelas no Supabase:

   cd backend
   npm run migrate

5. Inicie o sistema:

   npm start

6. Acesse no navegador:

   http://127.0.0.1:3000
