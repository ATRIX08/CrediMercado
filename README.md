# CrediMercado

Sistema local para controle de fiado em mercado.

## Como usar

Abra o arquivo `index.html` no navegador.

Login inicial:

- Usuario: `admin`
- Senha: `admin123`

## Recursos

- Login local.
- Cadastro e administracao de usuarios.
- Perfis de acesso.
- Cadastro de clientes.
- Lancamento de dividas.
- Registro de pagamentos com forma de pagamento.
- Painel financeiro.
- Lista de clientes com filtros.
- Cobranca pelo WhatsApp.
- Auditoria com busca por nome, acao e data.
- Exportacao e importacao de backup.

## Observacao

Esta versao roda direto no navegador e salva os dados no `localStorage` do computador.
